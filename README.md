# Life Expectancy Predictor: WHO Data-Driven Insights

## 📌 Project Overview
This project focuses on building a predictive regression model to estimate population life expectancy based on socio-economic and health indicators provided by the World Health Organization (WHO). The goal is to identify key drivers of longevity and provide data-driven support for policy-making.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Libraries:** Pandas, NumPy (Data Manipulation), Scikit-learn (Machine Learning), Matplotlib, Seaborn (Visualization)
* **Environment:** Jupyter Notebook

## 🚀 Key Deliverables
1. **Exploratory Data Analysis (EDA):** Performed correlation heatmaps, outlier detection, and distribution analysis to understand the dataset.
2. **Feature Engineering:** Developed a preprocessing pipeline to handle missing values and scale features.
3. **Model Selection:** Compared multiple regression models (Linear, Ridge, Lasso) using **RMSE** as the primary evaluation metric.
4. **Business Insights:** Translated model coefficients into actionable policy-relevant insights.

## 📊 Business Analyst Angle
This project demonstrates the ability to handle a full machine learning pipeline, from raw data cleaning to the extraction of strategic insights for decision support.
